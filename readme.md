Після виконання цієї роботи, результатом є те, що клонувавши віддалений репозиторій, 
додали до нього декілька нових папок та файлів, та виконали push із локального репозиторію 
на свій віддалений репозиторій.

Що таке DevOps?

DevOps - це методика, за допомогою якої відбувається взаємодія між розробниками та спеціалістів з 
інформаційно-технічного забезпечення, а після інтеграція її робочих процесів один в одного, 
щоб покращити якість продукту.
